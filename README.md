# Color Palette Generator

A simple web project that generates random color palettes and lets you copy hex values to the clipboard.

## Features

- Generate a palette of 5 random colors
- Click a color block or the copy icon to copy its hex code
- Responsive grid layout for desktop and mobile
- Clean, modern UI with smooth hover and click feedback

## Files

- `index.html` — page structure and palette layout
- `style.css` — styling for the app, including responsive design
- `script.js` — palette generation and clipboard interactions

## Usage

1. Open `index.html` in a web browser.
2. Click the **Generate Palette** button to create a new set of colors.
3. Click a color square or the copy icon to copy its hex value.

## Notes

- The app uses the browser Clipboard API, so it works best in modern browsers.
- No build tools or dependencies are required.
